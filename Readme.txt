Ubami
Ubami es un juego de disparos y plataformas inspirado en Bubble Bobble de 1998, desarrollado por un equipo de cinco personas.

Requisitos
Godot Engine: Versión 4.2.2 o superior.

Sistema Operativo: Windows, macOS, o Linux.

Instalación
Descargar Godot 4.2.2: Si aún no tienes Godot instalado, puedes descargarlo desde aquí.

Clonar el repositorio:

bash
git clone https://github.com/tu-usuario/Ubami.git
O simplemente descarga el archivo ZIP del repositorio y descomprímelo.

Ejecución
Abrir el proyecto en Godot:

Abre Godot Engine.

Haz clic en "Importar" y navega hasta la carpeta donde descargaste o clonaste Ubami.

Selecciona el archivo project.godot y haz clic en "Importar & Editar".

Ejecutar el juego:

En el editor de Godot, haz clic en el botón "Play" (el icono del triángulo verde) para ejecutar el juego.

Controles del Juego

Moverse: ASD

Disparar: j

Saltar: k
